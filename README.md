# nvi_test
# Тестовое задание

Необходимо разработать REST API сервис для работы с базой данных видео. Сервис является
простой оберткой над БД PostgreSQL и предоставляет методы для добавления, получения и
обновления информации о видео.

Инструкция по установке и настройке проекта

## Шаг 1. Клонирование репозитория

```bash
git clone git@github.com:GogaCpp/nvi_test.git
cd nvi_test
```

## Шаг 2. Настройка переменных окружения

Создайте `.env` - файл по [шаблону](https://github.com/GogaCpp/nvi_test/blob/master/.env.example)

```bash
cp .env.example .env
nano .env
```

## Шаг 3. Запуск

Для запуска введите команду:

```bash
make build
```

### Дополнительные команды

```bash
make stop                   # Остановка запущенного контейнера
make start                  # Запуск остановленного контейнера
make clean                  # Остановка контейнера, с удалением всех данных и кеша
make logs                   # Открыть логи контейнера
```
